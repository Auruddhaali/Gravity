# Gravity IDE

Gravity IDE is a custom fork of Visual Studio Code, designed and maintained by Auruddha Rushd Ali.

## The Repository

This repository contains the Gravity IDE source code - a customized version of Visual Studio Code with unique branding, themes, and features. This source code is available under the standard [MIT license](https://github.com/microsoft/vscode/blob/main/LICENSE.txt).

Gravity IDE is built on the foundation of Visual Studio Code, combining the simplicity of a code editor with powerful development tools. It provides comprehensive code editing, navigation, and understanding support along with lightweight debugging, a rich extensibility model, and lightweight integration with existing tools.

Gravity IDE features custom branding, a unique purple color scheme, and is tailored for an enhanced development experience.

## Building from Source

To build Gravity IDE from source:

```bash
npm install
npm run compile
```

For more detailed build instructions, please see the original [VSCode How to Contribute guide](https://github.com/microsoft/vscode/wiki/How-to-Contribute).
